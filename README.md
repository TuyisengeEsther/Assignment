# Esther. — Portfolio Website

Personal portfolio website for **Tuyisenge Esther**, Software Engineering Student at AUCA, Kigali, Rwanda.

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Introduction, bio, social links |
| About Me | `about.html` | Background, stats, skills |
| Projects | `projects.html` | Filterable project cards with modal |
